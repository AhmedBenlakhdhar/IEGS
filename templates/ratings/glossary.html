{% translate "1. Introduction & Purpose" %}

{% blocktranslate %}The Muslim Game Classification (MGC) system provides structured guidance for Muslims seeking to evaluate video games according to Islamic principles. Its primary purpose is to enable users to make informed decisions about game engagement, aligning their choices with the teachings of the Quran, the authentic Sunnah (Prophetic tradition), and the understanding of the As-Salaf As-Salih (Pious Predecessors).{% endblocktranslate %}

{% blocktranslate %}This document outlines the methodology used by MGC assessors to analyze games and assign ratings, ensuring consistency, transparency, and adherence to core Islamic values.{% endblocktranslate %}

{% translate "2. Core Principles" %}

  • {% translate "Foundation" %}: {% blocktranslate %}All assessments are grounded in the Quran and the authentic Sunnah, interpreted according to the understanding and consensus of the As-Salaf As-Salih. Where matters require ijtihad (scholarly reasoning), established principles of Fiqh (Islamic jurisprudence), such as Maqasid al-Shari'ah (Objectives of Islamic Law) and Sadd al-Dhara'i (Blocking the Means to Evil), are applied.{% endblocktranslate %}
  • {% translate "Objectivity & Consistency" %}: {% blocktranslate %}Assessors strive for objective analysis based on demonstrable game content and mechanics, applying the criteria consistently across all titles.{% endblocktranslate %}
  • {% translate "Scope" %}: {% blocktranslate %}The primary focus is the game's content, themes, mechanics, player actions, potential passive exposure, and potential impact (including normalization of values). Player behavior is generally outside the scope, unless directly necessitated or heavily incentivized by the game design.{% endblocktranslate %}
  • {% translate "Caution & Avoiding Doubt" %}: {% blocktranslate %}In cases of ambiguity or differing scholarly opinions on specific interpretations, a cautious approach is favored, often reflected in a 'Doubtful' rating or detailed notes.{% endblocktranslate %}
  • {% translate "Transparency" %}: {% blocktranslate %}The methodology aims for clarity. Justifications for ratings, especially the severity levels assigned to specific concerns and the resulting overall tier, are documented and made accessible.{% endblocktranslate %}

{% translate "3. The MGC Rating Process" %}

  1. {% translate "Game Selection & Initial Review" %}: {% blocktranslate %}Games are chosen based on factors like popularity, community requests, or potential Islamic relevance. An initial playthrough and research identify key themes and potential areas requiring detailed analysis.{% endblocktranslate %}
  2. {% translate "Detailed Content Analysis" %}: {% blocktranslate %}The game is systematically evaluated against the 20 Specific Content Concerns (Section 4), grouped into four main axes.{% endblocktranslate %}
  3. {% translate "Severity Assignment" %}: {% blocktranslate %}For each of the 20 concerns, a Severity Level (None, Mild, Moderate, Severe - see Section 5) is assigned. This assignment considers the presence, intensity, frequency, avoidability, player role, and nature (e.g., depiction vs. simulation vs. normalization) of the problematic content within that specific category.{% endblocktranslate %}
  4. {% translate "Overall Tier Calculation" %}: {% blocktranslate %}Based *only* on the assigned severity levels across the 20 concerns, the Overall MGC Rating Tier (Acceptable, Doubtful, Haram, Kufr/Shirk) is automatically calculated according to the strict logic defined in Section 6.{% endblocktranslate %}
  5. {% translate "Flag Assignment" %}: {% blocktranslate %}Relevant Content Flags (icons representing each concern) are *automatically assigned* to the game if the corresponding concern's severity level is Mild, Moderate, or Severe. These serve as a quick visual summary on the game detail page.{% endblocktranslate %}
  6. {% translate "Rationale & Documentation" %}: {% blocktranslate %}The overall reasoning for the final tier (based on the severity analysis) is summarized in the Summary & Rating Rationale field.{% endblocktranslate %}
  7. {% translate "Review & Publication" %}: {% blocktranslate %}Ratings undergo internal review for consistency and accuracy before being published. Ratings may be updated following significant game changes or new evidence.{% endblocktranslate %}

{% translate "5. Severity Level Definitions" %}

{% translate "For each specific concern, a severity level is assigned:" %}

{% translate "None" %} (N)
{% translate "No significant concerns identified." %}
{% translate "Mild" %} (L)
{% translate "Minor issues present, often optional, avoidable, or not normalized. Minimal risk." %}
{% translate "Moderate" %} (M)
{% translate "Noticeable concerns requiring player vigilance. May involve optional problematic elements, neutral normalization, or unavoidable minor issues." %}
{% translate "Severe" %} (S)
{% translate "Major concerns that are often required, unavoidable, pervasive, central, or positively promoted/normalized. High risk. Includes direct simulation/promotion of major sins or Kufr/Shirk." %}

{% translate "6. Overall MGC Rating Tier Calculation Logic" %}

{% translate "The final tier is calculated automatically based on the highest severity level found across all descriptors, according to their category mappings:" %}

  1. {% translate "Rule 1 (KUFR/SHIRK)" %}: {% blocktranslate %}If any descriptor in 'Risks to Faith' (Category A) is rated Severe, the Overall Rating is KUFR/SHIRK.{% endblocktranslate %}
  2. {% translate "Rule 2 (HARAM)" %}: {% blocktranslate %}Else, if any descriptor in 'Risks to Faith' (Category A) is rated Moderate OR any descriptor in 'Prohibition Exposure' (Category B) is rated Severe, the Overall Rating is HARAM.{% endblocktranslate %}
  3. {% translate "Rule 3 (DOUBTFUL)" %}: {% blocktranslate %}Else, if any descriptor in 'Risks to Faith' (Category A) is rated Mild OR any descriptor in 'Prohibition Exposure' (Category B) is rated Moderate OR any descriptor in 'Normalization Risks' (Category C) is rated Moderate or Severe OR any descriptor in 'Player Risks' (Category D) is rated Severe, the Overall Rating is DOUBTFUL.{% endblocktranslate %}
  4. {% translate "Rule 4 (ACCEPTABLE)" %}: {% blocktranslate %}If none of the above conditions (Rules 1, 2, 3) are met, the Overall Rating is ACCEPTABLE.{% endblocktranslate %}

{% translate "8. Disclaimer" %}

{% blocktranslate %}MGC ratings are provided as guidance based on ijtihad following the stated methodology. They are not legally binding fatwas (religious edicts) and do not represent the sole correct Islamic viewpoint on every nuanced issue. Variation in scholarly opinion exists on some matters. MGC aims to assist, not replace, individual responsibility before Allah. Ratings may be updated based on game changes or new evidence.{% endblocktranslate %}
