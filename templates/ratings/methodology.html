{% translate "1. Introduction & Purpose" %}

{% blocktranslate %}The Muslim Game Classification (MGC) system provides structured guidance for Muslims seeking to evaluate video games according to Islamic principles. Its primary purpose is to enable users to make informed decisions about game engagement, aligning their choices with the teachings of the Quran, the authentic Sunnah (Prophetic tradition), and the understanding of the As-Salaf As-Salih (Pious Predecessors).{% endblocktranslate %}

{% blocktranslate %}This document outlines the methodology used by MGC assessors to analyze games and assign ratings, ensuring consistency, transparency, and adherence to core Islamic values.{% endblocktranslate %}

{# 2. Core Principles Card #}

{% translate "2. Core Principles" %}

  • {% translate "Foundation" %}: {% blocktranslate %}All assessments are grounded in the Quran and the authentic Sunnah, interpreted according to the understanding and consensus of the As-Salaf As-Salih. Where matters require ijtihad (scholarly reasoning), established principles of Fiqh are applied.{% endblocktranslate %}
  • {% translate "Objectivity & Consistency" %}: {% blocktranslate %}Assessors strive for objective analysis based on demonstrable game content and mechanics, applying the criteria consistently across all titles.{% endblocktranslate %}
  • {% translate "Scope" %}: {% blocktranslate %}The primary focus is the game's content, themes, mechanics, player actions, and potential impact (including normalization). Player behavior is generally outside the scope unless necessitated by game design.{% endblocktranslate %}
  • {% translate "Caution & Avoiding Doubt" %}: {% blocktranslate %}In cases of ambiguity or differing scholarly opinions, a cautious approach is favored, often reflected in a 'Doubtful' rating or detailed notes.{% endblocktranslate %}
  • {% translate "Transparency" %}: {% blocktranslate %}The methodology aims for clarity. Justifications for ratings and severity levels are documented and made accessible.{% endblocktranslate %}
{# 3. The MGC Rating Process Card #}

{% translate "3. The MGC Rating Process" %}

  1. {% translate "Initial Review" %}: {% blocktranslate %}Games are selected, played, and researched to identify key themes and potential areas for analysis.{% endblocktranslate %}
  2. {% translate "Content Analysis" %}: {% blocktranslate %}Systematic evaluation against the 20 Specific Content Concerns (Section 4).{% endblocktranslate %}
  3. {% translate "Severity Assignment" %}: {% blocktranslate %}Assigning a Severity Level (None, Mild, Moderate, Severe - see Section 5) to each of the 20 concerns based on presence, intensity, avoidability, etc.{% endblocktranslate %}
  4. {% translate "Tier Calculation" %}: {% blocktranslate %}The Overall MGC Rating Tier (Acceptable, Doubtful, Haram, Kufr/Shirk) is automatically calculated based on the logic in Section 6.{% endblocktranslate %}
  5. {% translate "Flag Assignment" %}: {% blocktranslate %}Relevant Content Flags (icons) are automatically assigned if a concern's severity is Mild, Moderate, or Severe.{% endblocktranslate %}
  6. {% translate "Rationale & Documentation" %}: {% blocktranslate %}The reasoning for the final tier is summarized.{% endblocktranslate %}
  7. {% translate "Review & Publication" %}: {% blocktranslate %}Internal review before publishing; ratings may be updated.{% endblocktranslate %}
